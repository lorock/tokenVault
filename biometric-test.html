③ 运行日志

如何在本机/真机运行（WebAuthn 需要安全上下文）

本机（桌面 Chrome/Edge 带指纹或 Windows Hello）：

npm run dev 后浏览器打开 http://localhost:5173/biometric-test.html（localhost 视为安全上下文）。

手机（iPhone 面容 / 安卓指纹）：WebAuthn 必须 HTTPS。把本机 dev server 暴露成 HTTPS：

npx vite --host + 隧道：cloudflared tunnel --url http://localhost:5173（或 ngrok），用返回的 https 链接在手机打开本页。

⚠️ http://192.168.x.x:5173 等非安全上下文会被浏览器拒绝 PRF，无法联调。
